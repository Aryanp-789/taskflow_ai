📌 TaskFlow AI – SaaS Landing Page

A modern, responsive SaaS landing page built using React (Vite) + Tailwind CSS.
TaskFlow AI is a fictional productivity platform that helps teams manage tasks, improve workflow efficiency, and boost productivity using AI-powered features.

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/taskflow_ai.git
2. Navigate to project folder
cd taskflow_ai
3. Install dependencies
npm install
4. Run development server
npm run dev
📦 Build for Production
npm run build
